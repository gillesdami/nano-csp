# async generator

Generators that return promises, when a promise resolve it returns an object {value, done}

ES2018

Already in:
node harmony
Firefox 57
Chrome 63
Safari 11.1

//SHEMA async gen

## basic usage

async function* agen() {
    yield 1;
    yield 2;
    yield fetch("something");
}

asyncIterator = agen();
asyncIterator.next().then(({ value, done }) => /* ... */);

for await (const line of readLines(filePath)) {
  console.log(line);
}
